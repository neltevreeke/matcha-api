# Matcha - api

## Getting started

```shell
npm install
cp .env.example .env
```

## Seeding the database

```shell
npm run seed
```

## Running development environment

```shell
npm run dev
```

## Creating a component

To easily create a scaffolds of components etc. we use Plop.

Please make sure you install Plop first: `npm install plop --g`

```shell
plop component
```
